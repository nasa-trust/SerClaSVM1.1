GenoPro
GenoPro is a genealogy software for drawing family trees. With GenoPro, users can view complete graphical representation of their genealogy tree and can see both ancestors and descendants at once. The API exposes the functionality of the software to developers. Many methods are exposed including file delete, publish report, search city, collaboration projects and more. The API uses SOAP calls and responses are formatted in XML.
