Wireframe and prototyping software
Pidoco
Pidoco is a web based GUI design software for creating wireframes. The pidoco API allows developers to include pidoco prototypes into other third party applications or create their own tools that use pidoco. Functionality includes working with protoypes, pages within the prototypes, layes and discussions for commenting on prototypes. The API uses RESTful calls with responses formatted in JSON.
