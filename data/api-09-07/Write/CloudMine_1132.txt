Backend as a service platform
CloudMine
CloudMine is a backend-as-a-service platform that removes the need for mobile app developers to build fully custom backend solutions for their applications. It offers a set of RESTful APIs that allows developers to create backend solutions for their applications. Solutions include schema-free data structure storage, user account creation and management, and server-side business logic. Responses are formatted in JSON.
