Search engine promotion valuation service
SEOBudget
SEOBudget is a Russian site for assessing the value of your search engine links based on traffic, affiliate connections, and other metrics. The SEOBudget API follows the XMl-RPC protocol and allows the user to access site ranking data, check for mirror sites in the RU domain, determine site traffic and index pages among other things.
