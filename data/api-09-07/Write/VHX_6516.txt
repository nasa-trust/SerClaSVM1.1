VHX
VHX is a service that brings online videos together from around the web. Users can discover, share and watch videos as well as create their own personal dashboards. The VHX API lets users access their accounts via other web sites or applications. The API can be used to read and write videos to accounts, create video playlists and embed a player into a site. It uses RESTful calls and responses are formatted in XML and JSON.
