IP address lookup service
Trackip 
Trackip is a German site that tracks user IP addresses and provides the geographical location based on the IP. The API gives developers programmatic access to the site functionality and data for use in their own applications or websites. The API lets users look up the internet host, IP address, city, country, latitude and longitude. It uses RESTful calls and responses are formatted in XML. The site and documentation are in German.
