ABA key to routing number validation service
Quentin Sager ABA Routing Number
This service provides ABA key to routing number validation in the form of ABA checksum validation and database look-up. When a database match is found, the service will identify whether the routing number has been retired and, if appropriate, return the new number. It will also return the bank or institution name the number is assigned to and the location of the primary office using the number.
