Globus Online Transfer
Globus Online is a file transfer service designed specifically for researchers requiring access to files located on multiple devices. The REST-based Transfer API allows users to perform a number of Globus Online&#039;s file transferring functions programmatically. REST-accessible functions include submitting file transfer tasks, monitoring the progress of those tasks, managing file transfer endpoints, and listing remote directories.
