Topic Maps Browser and Editor
Maiana
The Maiana API allows users to store, query and modify their data in Maiana. Maiana is a social explorer for structured data. Maiana is a service to host, explore and share Topic Maps sources. The API allows users to create, update, delete and download topic maps, show lists of topic maps and more. It uses RESTful protocol and responses are formatted in JSON.
