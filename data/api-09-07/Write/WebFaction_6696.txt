WebFaction
The WebFaction API is a XML-RPC interface for managing many control panel and account tasks. With the WebFaction API, users can automate application installation, email address configuration, and more. WebFaction is an internet hosting service. The WebFaction API allows users to write scripts to automate certain tasks that would normally be accomplished via the control panel or SSH. The API is a set of methods available via XML-RPC calls.
