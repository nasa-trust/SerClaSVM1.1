Acquia Cloud
Acquia is a commercial open source software company that provides products and services for Drupal. Acquia&#039;s cloud based solution lets users create Drupal based web sites. The Acquia Cloud API allows developers to extend and customize Acquia Cloud. The API allows direct control of Acquia Cloud sites and supports developer workflow, database, and backup management. It uses RESTful calls and responses are formatted in JSON.
