CAD model viewing, printing and collaboration
Autodesk Freewheel
From their site: Autodesk Freewheel is an Ajax (Asynchronous JavaScript and XML) based web service/application built to provide CAD model viewing, printing and collaboration, through the dynamic rendering of DWF file data and graphics information. You can think of Freewheel as a set of componentized Web Services and JavaScript frameworks. Together these components make up the Interactive Ajax DWF Viewer.
