Blog Management Tool
Skyrock
Skyrock offeres an XML-RPC API that lets developers create apps that can manage their blogs. The API supports methods that interact with blogs on Blogger, metaWeblog, and Skyrock. For Skyrock partners, and API Key is used for authentication. For other developers and non-partners, a username/password must be sent with each API call, or a session string must be created.
