Affiliate and merchant transaction information retrieval service
Affiliate Window ProductServe
Affiliate Window is an affiliate marketing company and the largest independent network in the UK with over 750 merchants. The ProductServe API provides access to Affiliate Window product data to allow development of products such as the ShopWindow Toolkit. With the API, developers can query and retrieve lists of products, merchants, discount codes and more. The API uses SOAP calls and responses are formatted in XML.
