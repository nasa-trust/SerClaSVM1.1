Online background check service
Reppify
Reppify offers an online background screening service. The Reppify API enables developers to interact with the Reppify web site programmatically. This API is designed to enable users to integrate Reppify with other applications. It can be used to manage the reppify work-flow by creating jobs (called UserLists in the web API), add candidates (called users in the Reppify API) and send invitations to request that candidates score themselves. The API uses RESTful calls and responses are formatted in XML.
