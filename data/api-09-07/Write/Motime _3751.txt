Motime 
The Motime Open Access platform is an affiliate program based on the APIs of the Motime service, that allows partners/affiliates to advertise Motime content on their own web and/or mobile sites and earn money for each referral given to Motime.

The search API uses the REST protocol to allow developers to link their content with Motime&#039;s catalog of ringtones.
