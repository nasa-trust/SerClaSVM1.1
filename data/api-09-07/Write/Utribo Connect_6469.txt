Utribo Connect
Utribo Connect, is software-as-a-service that offers click-to-call and click-to-video. Businesses using Utribo can route web users to a live person using voice or video through the use of a link that appears as a button on the web page. Users can try Connect for free, however API access requires a monthly fee. The API can be used to connect Utribo Connect to a telecom backend. Documentation is not publicly available.
