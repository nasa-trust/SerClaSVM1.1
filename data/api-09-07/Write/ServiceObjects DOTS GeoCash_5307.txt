ATM location service
ServiceObjects DOTS GeoCash
The ServiceObjects DOTS GeoCash API provides users with a collection of SOAP-based calls for locating ATMs. These calls can return ATM locations for a given five-digit ZIP code, for a street name within a given ZIP code, for a given city and two-letter state abbreviation, or within a specified radius (in miles) of a given ZIP code.
