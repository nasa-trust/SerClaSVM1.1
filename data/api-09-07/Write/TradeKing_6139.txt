TradeKing
TradeKing is an online stock and options broker. It offers secure trading API for stock and options trading. The API offers the functionality of the web based service including account management, trading activity including order submission and modification, and access to real-time and delayed market data. The API is publicly available to all TradeKing customers. Full documentation is available at http://developers.tradeking.com.
