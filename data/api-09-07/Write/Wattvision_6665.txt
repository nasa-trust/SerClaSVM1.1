Home energy measurement service
Wattvision
The Wattvision API allows developers to create products that are wattvision-enabled. Wattvision is a service that gives users a live view of their energy use on a computer or mobile device. Wattvision works through a sensor that connects to a home&#039;s electricity meter and wireless internet connection. The Upload API allows users to upload their home&#039;s energy use data to wattvision&#039;s servers, for storage, comparison, sharing, and visualization. The Download API allows user access the data for their house. The API uses RESTful protocol and responses are formattted in JSON.
