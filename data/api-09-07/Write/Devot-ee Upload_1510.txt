Devot:ee Upload
Devot:ee is a collaborative community website for fostering growth and knowledge-sharing about ExpressionEngine. ExpressionEngine is a budding content management system. On Devot:ee, developers can plans or information about new plug-ins, extensions, modules, and more that they are developing. The Upload API allows developers upload files and release data about their add-ons. It is a RESTful API that returns JSON formatted data.
