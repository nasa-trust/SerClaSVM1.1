Online source of DNS tools
ViewDNS.info
ViewDNS in an online source for DNS related tools. The tools provided by ViewDNS.info allow users to gather a large amount of data about a given website or IP address. The ViewDNS.info API allows webmasters to integrate the tools provided by ViewDNS.info into their own sites. Response formats include XML and JSON. Access to the ViewDNS.info API is free for a limited time, developers should contact the provider to gain access to the API and the associated documentation.
