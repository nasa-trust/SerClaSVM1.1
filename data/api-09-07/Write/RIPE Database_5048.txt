RIPE Database
The RIPE NCC is one of five Regional Internet Registries (RIRs) providing Internet resource allocations, registration services and coordination activities that support the operation of the Internet globally. The RIPE Database contains registration information for networks in the the RIPE NCC service region and related contact details. The RIPE Database RESTful API provides an interface to the Whois Database and offers developers methods to programmatically manipulate RIPE Database objects. API calls are responded to in XML and JSON format.
