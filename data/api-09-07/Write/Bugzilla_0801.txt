Bugzilla
Bugzilla is the bug tracking and reporting system created and used by Mozilla. It is also available for use by other projects and organizations. The life cycle of a bug, also known as workflow, is customizable to match the user&#039;s needs. The Bugzilla REST API provides users with programmatic access Bugzilla&#039;s service using either JSON or JSONP.
