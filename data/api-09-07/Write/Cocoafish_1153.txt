Backend service for mobile applications
Cocoafish
Cocoafish is a backend service that allows developers to create mobile applications without writing additional code. Cocoafish provides network features and data objects for applications and the ability to feature numerous API functions in one interface.

Cocoafish is in private beta; interested developers should sign up here: http://cocoafish.com/beta/signup.
