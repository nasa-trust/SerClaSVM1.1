Campaign donation and vote tracking service
MAPLight.org Bill Positions
MAPLight.org performs research on supporting and opposing interests for legislative bills. MAPLight.org&#039;s Bill Positions API provides a tool for discovering the connection between campaign donations and legislative votes. It can also be mashed up with other databases that contain information about bills or politically-active organizations. This RESTful API returns results in JSON or XML.
