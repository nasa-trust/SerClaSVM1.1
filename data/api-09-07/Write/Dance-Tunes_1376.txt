Dance-Tunes
Dance-Tunes is a site that specializes in mp3 downloads for all kinds of dance music. Users can search the site database for artists, labels, tunes, mixes and albums. The API serves as an interface between the database and the client front-end. Data that can be returned by the API includes artists, labels, genre, charts and more. It uses RESTful calls and responses are formatted in XML.
