Cron job scheduling service
CloudBlocks CloudQuartz
CloudQuartz by CloudBlocks lets users define and schedule their jobs to run at specified times and on different servers. The jobs can then be monitored and controlled centrally via the web. The API allows users to create and schedule jobs, fetch new jobs when they are due to run and notify the server of agent status. The API uses RESTful calls and responses are formatted in JSON.
