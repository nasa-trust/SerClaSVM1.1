Pygame
The API provides both a RESTful and XML-RPC interface to pygameweb.no-ip.org. Pygame is a set of Python modules designed for writing video games. The API allows users to access data from pygameweb.no-ip.org for games and other programs. The Pygame API lets users read and edit projects and releases. Responses can be formatted in XML, JSON, YAML, Pickle and Django.
