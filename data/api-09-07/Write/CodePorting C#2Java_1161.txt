CodePorting C#2Java
CodePorting is a free source code converter that helps users conver their C# files into Java. Using CodePorting, developers can make their .NET applications cross platform compatible and  can migrate their .NET solutions, projects and files into Java in the cloud. The API allows developers to build their own customized code conversion applications. The API exposes functionality relating to snippet porting, code analysis, library and project management. It uses RESTful calls and responses are formatted in XML.
