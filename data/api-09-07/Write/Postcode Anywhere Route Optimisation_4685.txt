Postcode Anywhere Route Optimisation
Postcode Anywhere is an information-as-a-service provider offering over 250 separate web services covering a broad range of business efficiency tools for address and data capture, validation and cleansing to geo-coding, mapping, and route planning and optimization.

The Route Optimisation allows users to optimise routes for multiple vehicles and drop-off points. It can return the directions between two or more points and pre validate route locations.

The API uses a number of protocols including REST, SOAP and XML-RPC. Responses can be formatted in XML, JSON, CSV, HTML and PDF.
