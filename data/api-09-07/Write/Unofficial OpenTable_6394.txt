Restaurant reservation service
Unofficial OpenTable
Open table is an online restaurant reservation service. With the service users can make online reservations, read restaurant reviews from other diners, and earn points towards free meals. This API allows developers to access the data from OpenTable. Data includes finding single or multiple restaurants, along with information about that restaurant and URLs where reservations can be made. The API uses RESTful calls and responses are formatted in JSON.
