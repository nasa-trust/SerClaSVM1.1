Search and browse Twitter lists
TLists
The TLISTS API enables filtering of off-topic tweets to provide users the best real-time users and lists for any topic. TLISTS provides real-time ?Channels? - live streams of relevant messages from respected sources on a particular topic. TLISTS crawls more than half a million Twitter lists to identify the most respected sources on Twitter. The API uses RESTful calls and responses are formatted in either XML or JSON. No authentication is required.
