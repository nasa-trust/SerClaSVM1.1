NetDNA provides a high-end SSL that provides online security without sacrificing performance. The NetDNA API is an XML-RPC based programming interface for many of the most requested functions found in the NetDNA Control Panel. The API includes functions to create, modify, and delete zones as well as prime and purge cache files. It can also list and update users, and create all of the various reports.
